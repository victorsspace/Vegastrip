SETUP GUIDE
Follow these 4 steps on your phone. Takes ~15 minutes. Everything is free!
1 Create Firebase Project
Go to console.firebase.google.com
→ Add project → Name it "vegas-trip" → Create
→ Authentication → Get started → Email/Password → Enable
→ Firestore Database → Create database → Production mode
2 Get Your Config Keys
Firebase Console → Project Settings (⚙️ gear icon)
→ Your apps → Add app → Web (</>) → Register
→ Copy the values from the firebaseConfig object shown
3 Edit This HTML File
Open this file in a text editor app on your phone:
→ iPhone: Textastic or Koder (free)
→ Android: Acode or QuickEdit (free)

Find the CONFIG block at the very top of the file and fill in:
superAdminEmail: "[email protected]"
firebase: {
  apiKey: "AIza...",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123...:web:abc..."
}
4 Add Firestore Security Rules
Firebase Console → Firestore → Rules tab
Replace everything with:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /users/{uid} {
      allow create: if request.auth.uid == uid
        && request.resource.data.role == 'user'
        && request.resource.data.status == 'pending';
      allow read: if request.auth != null;
      allow update,delete: if get(/databases/$(db)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /trip/{doc=**} {
      allow read,write: if get(/databases/$(db)/documents/users/$(request.auth.uid)).data.status == 'approved';
    }
  }
}
Then click Publish.

Finally → Upload this HTML to app.netlify.com/drop and share the link!
5 Register — You're Auto Super Admin!
Open your site → click Register → use the same email you set as superAdminEmail

✅ You will be instantly approved as Admin — no waiting, no approval needed.
Everyone else who registers will wait for your approval in the Admin panel.

✅ Netlify Free Hosting — unlimited static site
✅ Total cost — $0 forever for a friend group
